### Adding Todos

Allows user to add tasks without specified time.

**Example:** `todo homework`

A task with the name "homework" will be added to the list. 

```
[T][] homework 
```

### Adding Deadlines

Allows user to add tasks with specified deadline.

**Example:** `deadline assignment `

A task with the name "homework" will be added to the list.

```
[T][] homework 
```
